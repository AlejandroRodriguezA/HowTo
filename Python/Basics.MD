## Crear y activar un ambiente (environment)

```bash
	$ python -m venv env/       # Your binary is maybe called 'python3'
	$ source env/bin/activate   # for Linux and macOS
	$ env\Scripts\activate.bat  # for Windows
```

**Nota:** Para desactivarlo usar:   

```bash
deactivate
```
